# 电视剧电影推荐系统 python197

本项目是基于Python Django框架开发的电视剧和电影推荐系统，使用MySQL作为数据库管理系统，设计理念围绕用户需求进行。

## 项目概述


## 技术栈

- 后端：Python Django
- 数据库：MySQL
- 前端：HTML/CSS/JavaScript（根据提供的信息，前端技术栈未明确，此处假设使用常规前端技术）

## 功能模块

- 用户注册与登录
- 用户行为记录
- 内容推荐算法
- 数据库管理

## 系统角色

- 普通用户：浏览内容，获得推荐，记录观看历史
- 管理员：管理系统内容，查看用户行为数据

## 运行环境

- Python 3.x
- Django 2.x
- MySQL 5.x

1. 安装Python环境和Django框架
2. 安装并配置MySQL数据库
3. 导入数据库结构及数据
4. 运行Django项目
